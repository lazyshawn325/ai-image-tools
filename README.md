## 🎨 视觉美学：Deep Cosmos & Bento Grid

本项目不仅是一个工具集合，更是一场视觉盛宴。我们采用了前卫的 **“深邃宇宙 (Deep Cosmos)”** 设计语言：

- **Bento Grid 布局**: 苹果风格的非对称模块化设计，节奏感十足。
- **3D Tilt 交互**: 每一个工具卡片都具备物理感应的 3D 倾斜效果，光随人动。
- **Mesh Gradients**: 丝滑的网格渐变背景，营造沉浸式的高级质感。
- **玻璃态 (Glassmorphism)**: 极致的磨砂玻璃效果与流光边框。

## 🚀 核心功能

### 🛠️ 生产力工具
- **智能图片压缩**: 毫秒级压缩，大幅减小体积而不损失画质。
- **专业级裁剪**: 引入 `react-easy-crop`，支持 360° 旋转、无极缩放和多种预设比例。
- **批量格式转换**: PNG, JPEG, WebP, GIF 互转，支持任务队列。
- **AI 智能去背景**: 纯前端运行 AI 模型，一键移除背景，发丝级精度。
- **无损画质放大**: 采用先进算法实现 2x/3x/4x 像素增强。

### 🛡️ 隐私与安全
- **100% 本地处理**: 所有操作均在浏览器内存中完成，**图片永不上传服务器**。
- **隐私保护**: 彻底杜绝数据泄露，断网亦可使用。

## ⚡ 技术亮点

- **Next.js 15 (App Router)**: 利用最新的服务器组件与客户端组件混合模式。
- **Tailwind CSS 4**: 极简配置，极高性能。
- **Framer Motion**: 电影级的交互动画曲线。
- **PWA 支持**: 可安装到桌面或手机，像原生 App 一样流畅。
- **i18n 多语言**: 完整的中英双语支持，包含自动化 SEO 矩阵。
